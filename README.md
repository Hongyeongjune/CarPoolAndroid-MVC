# Carpool
카풀 어플리케이션 안드로이드

## 서비스 소개
운전자가 카풀 등록을 해놓으면 사용자는 등록된 카풀 목록을 확인 후 카풀 신청을 해 연결 후 카풀을 하게 되는 서비스

## 구현사항

###Front end

**Android Studio**
1. MVC 패턴을 활용하여 개발
2. NetRetrofit 라이브러리를 사용하여 SpringBoot와 통신
3. Google Map API를 사용하여 지도를 활용
4. T-Map API를 사용하여 최단거리 경로 안내
5. FCM API를 사용하여 어플간 알림 메시지 전송
